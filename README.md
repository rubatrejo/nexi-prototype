# NEXI — Hotel Self Check-in Kiosk

Interactive prototype for NEXI, a self-service kiosk product by [TrueOmni](https://trueomni.com) for the hospitality industry.

**Live Demo:** [nexi-prototype.vercel.app](https://nexi-prototype.vercel.app)

## Overview

NEXI transforms hotel lobbies with a touchscreen kiosk that handles check-in, check-out, room service, wayfinding, events, bookings, and more — 16 modules working together to improve guest experience and operational efficiency.

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- Framer Motion
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [localhost:3000](http://localhost:3000)

## Features

- 16 kiosk modules with 78+ screens
- Light and dark mode
- 4 languages (EN, ES, FR, JA)
- Onboarding flow with ROI calculator
- Cinematic photography-driven design
- Touch-optimized interface

## Design

Nordic Cinematic style — Mona Sans + Inter typography, photography-forward, film grain texture, glass card effects. See `docs/` for full design documentation.
